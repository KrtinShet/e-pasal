# E06-S02: Core Components

## Summary

Port and implement core UI components from Minima MUI patterns, rewritten with Tailwind CSS and design tokens.

## Why

- Provides reusable, consistent components across all apps
- Reduces duplication and maintenance burden
- Components automatically respond to theme token changes
