# E04-S08 Acceptance Criteria

- [ ] Merchants can configure payment providers from dashboard settings.
- [ ] Credentials are stored encrypted at rest (AES-256).
- [ ] API responses mask credential values (\*\*\*\*abcd format).
- [ ] Storefront checkout shows only enabled payment methods for the store.
- [ ] Payment lifecycle (initiate -> callback -> verify) updates order status automatically.
- [ ] Dashboard integration settings page has both Payments and Logistics tabs.
